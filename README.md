# 📚 RAG Chatbot using LangChain, FAISS & Gemini

## Overview
This project is a Retrieval-Augmented Generation (RAG) Chatbot that allows users to upload PDF documents and ask questions in natural language. The chatbot retrieves relevant information from the document using vector search and generates accurate answers using Google's Gemini model.

## Features
- PDF document ingestion
- Text chunking and preprocessing
- Semantic search using FAISS
- HuggingFace embeddings
- Google Gemini integration
- Source page references
- Interactive question answering

## Tech Stack
- Python
- LangChain
- FAISS
- HuggingFace Embeddings
- Google Gemini API
- PyPDF
- Google Colab

## Workflow
1. Upload PDF
2. Extract text
3. Split into chunks
4. Generate embeddings
5. Store vectors in FAISS
6. Retrieve relevant chunks
7. Generate answers using Gemini

## Example Questions
- What is backpropagation?
- Summarize the document.
- Explain neural networks.
- What are the main topics covered?
